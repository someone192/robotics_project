Проект представляет собой простейший прототип робота для расстановки грузов с автоматической оптимизацией размещения. Текстовый файл - пояснительная записка, приложенная к роботу на олимпиаде по технологии, описывает главные характеристики и принцип работы устройства. Приложенный код - простой механизм распределения, работающий на чистом python, код с использованием C++ и boost находится в разработке, но еще сырой. Работает устройство с применением ESP 32, простейшей печатной платы собственной разработки. В ходе работы были попытки использования openCV(yolov8, затем yolov11), но на момент написания еще не интегрировано в систему. Основная идея проекта - снять с людей не только физическую нагрузку, но и необходимость расчетов положения, добившись полной автоматизации.
